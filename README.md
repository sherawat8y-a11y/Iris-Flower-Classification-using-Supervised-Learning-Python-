# 🌸 Iris Flower Classification using Machine Learning
##Overview
This project is a supervised machine learning model that classifies iris flowers into different species based on:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
The model is trained on the famous Iris dataset and predicts the species accurately.
##Dataset
- Iris Dataset (built-in dataset from scikit-learn)
- 3 classes:
  - Setosa
  - Versicolor
  - Virginica
##Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Visualization of sepal and petal features
- Model training using multiple algorithms
- Performance evaluation
##Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
--
##Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
---
## Results
The model achieved high accuracy in classifying iris species.  
Petal length and petal width were found to be the most important features.
